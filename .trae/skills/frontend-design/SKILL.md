# 前端设计技能

本技能指导创建独特、生产级的前端界面，避免千篇一律的"AI堆砌"美学。以卓越的美学细节和创意选择实现真正可运行的代码。

## 设计思维

在编码之前，理解上下文并确定一个醒目美学方向：

**目的：** 这个界面解决什么问题？谁会使用它？

**风格：** 选择一个极致方向：
- 极度极简
- 极繁主义
- 复古未来主义
- 有机自然
- 奢华精致
- 趣味玩偶风
- 杂志编辑风
- 粗野/raw
- 艺术装饰/几何
- 柔和粉彩
- 工业实用主义

**约束：** 技术要求（框架、性能、无障碍访问）。

**差异化：** 是什么让它令人难忘？有什么是人们会记住的？

关键：选择清晰的概念方向并精确执行。极繁主义和精致极简都可以——关键是刻意性，而非强度。

## 实现标准

实现可工作的代码（HTML/CSS/JS、React、Vue等），需要：

- 生产级且功能完整
- 视觉上令人惊叹且令人难忘
- 具有清晰美学观点的一致性
- 每个细节都精心打磨

## 前端美学指南

### 字体排版
- 选择美观、独特、有趣的字体
- 避免 Arial、Inter、Roboto 等通用字体
- 选择能提升前端美学的独特选择
- 将独特的展示字体与精致的正文字体搭配

### 颜色与主题
- 坚持一致的美学风格
- 使用 CSS 变量保持一致性
- 主导演颜色配合锐利的强调色，优于平淡、均匀分布的配色方案

### 动效
- 使用动画实现效果和微交互
- 优先使用纯 CSS 解决方案
- React 可用 Motion 库
- 聚焦高影响时刻：一次精心编排的页面加载配合交错揭示（animation-delay），比分散的微交互更能创造愉悦感
- 使用滚动触发和令人惊喜的悬停状态

### 空间构成
- 出人意料的布局
- 不对称
- 重叠
- 对角线流动
- 打破网格的元素
- 大量负空间或受控密度

### 背景与视觉细节
- 创造氛围和深度，而非默认纯色
- 添加与整体美学匹配的上下文效果和纹理
- 应用创意形式，如：
  - 渐变网格
  - 噪点纹理
  - 几何图案
  - 叠加透明度
  - 戏剧性阴影
  - 装饰性边框
  - 自定义光标
  - 颗粒叠加

## 应避免的反模式

绝对不要使用：
- 通用AI生成美学
- 过度使用的字体族（Inter、Roboto、Arial、系统字体）
- 陈词滥调的配色方案（尤其是白底紫渐变）
- 可预测的布局和组件模式
- 缺乏上下文特异性的千篇一律设计
- 代际相传的常见选择如 Space Grotesk

## 设计执行

创造性地解读，做出真正为上下文量身定制的意外选择。每个设计都应该独一无二。在浅色和深色主题、不同字体、不同美学之间变化。

重要：根据美学愿景匹配实现复杂度：
- 极繁设计需要精心制作的代码，包含大量动画和效果
- 极简或精致设计需要克制、精确，以及对间距、字体排版和微妙细节的精心关注

优雅源于执行愿景的能力。

记住：Claude 能够创造非凡的创意作品。不要保守，展示真正突破框框、全力以赴投入独特愿景时能创造的东西。
